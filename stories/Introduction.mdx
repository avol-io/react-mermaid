# @avol-io/react-mermaid

A React component that renders **[Mermaid](https://mermaid.js.org/)** diagrams from text input,
with built-in **zoom, pan, directional controls**, **fullscreen**, **download** (SVG, .mmd),
**open in draw.io**, and an **error modal** for invalid syntax.

---

## Installation

```bash
npm install @avol-io/react-mermaid
```

### Peer dependencies

This package ships **without** its core dependencies bundled. Install them alongside:

```bash
npm install react react-dom mermaid react-zoom-pan-pinch
```

| Peer dependency | Required version |
|---|---|
| `react` | `>=19` |
| `react-dom` | `>=19` |
| `mermaid` | `>=11` |
| `react-zoom-pan-pinch` | `>=3` |

---

## Quick start

```tsx
import { MermaidDiagram } from '@avol-io/react-mermaid'

export function App() {
  return (
    <MermaidDiagram
      chart={`
        graph TD
          A[Start] --> B{Decision}
          B -- Yes --> C[Done]
          B -- No  --> D[Skip]
      `}
      width="100%"
      height="400px"
    />
  )
}
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `chart` | `string` | required | Mermaid source text |
| `width` | `string` | `"100%"` | CSS width of the container |
| `height` | `string` | `"400px"` | CSS height of the container |
| `config` | `MermaidConfig` | — | Options forwarded to `mermaid.initialize()` |
| `enableDownloadSvg` | `boolean` | `true` | Show "SVG" option in the download dropdown |
| `enableDownloadMmd` | `boolean` | `true` | Show ".mmd" option in the download dropdown |
| `enableDrawio` | `boolean` | `true` | Show "Open in draw.io" button in the toolbar |

> The download dropdown icon is automatically hidden when both `enableDownloadSvg`
> and `enableDownloadMmd` are `false`.

---

## Toolbar

### Top-right
- **Download dropdown** — exports the diagram as SVG or `.mmd` source (each option individually toggleable)
- **Open in draw.io** — opens the diagram directly in [app.diagrams.net](https://app.diagrams.net) using the Mermaid import URL
- **Fullscreen** — toggles fullscreen overlay

### Bottom-right (3×3 grid)
- Arrow buttons for panning in all four directions
- Center reset button — fits the diagram back to the container
- Zoom in / Zoom out

## Styling

No CSS import needed. On first render a single `<style id="avol-react-mermaid-styles">` tag is
injected into `<head>` and reused by all instances.

All colors and sizes are exposed as **CSS custom properties** on `:root`. Override them globally
or scope them to a specific instance:

```css
/* global theme override */
:root {
  --mermaid-toolbar-bg: rgba(30, 30, 30, 0.85);
  --mermaid-toolbar-color: #f9fafb;
  --mermaid-toolbar-hover-bg: #1f2937;
  --mermaid-border-color: #374151;
  --mermaid-border-radius: 12px;
  --mermaid-fullscreen-bg: #111827;
}
```

| Token | Default | Controls |
|---|---|---|
| `--mermaid-bg` | `transparent` | container background |
| `--mermaid-border-color` | `rgba(0,0,0,0.12)` | container border |
| `--mermaid-border-radius` | `8px` | container corner radius |
| `--mermaid-toolbar-bg` | `rgba(255,255,255,0.9)` | button background |
| `--mermaid-toolbar-color` | `#374151` | button icon color |
| `--mermaid-toolbar-hover-bg` | `#ffffff` | button hover background |
| `--mermaid-toolbar-focus-ring` | `#01696f` | focus outline color |
| `--mermaid-toolbar-btn-size` | `30px` | button width & height |
| `--mermaid-dropdown-bg` | `#ffffff` | dropdown panel background |
| `--mermaid-dropdown-item-hover` | `#f3f4f6` | dropdown item hover |
| `--mermaid-error-bg` | `#fef2f2` | error banner background |
| `--mermaid-error-color` | `#dc2626` | error banner text |
| `--mermaid-fullscreen-bg` | `#ffffff` | fullscreen overlay background |
